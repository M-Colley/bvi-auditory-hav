# Effects of Auditory Information for People With Visual Impairments in Highly Automated Vehicles

This repository contains the study data, the R analysis script, and the generated figures for the paper:

> Mark Colley, Tobias Aescht, Omid Rajabi, Max Rädler, Pascal Jansen, and Enrico Rukzio. 2026. **Effects of Auditory Information for People With Visual Impairments in Highly Automated Vehicles.** [TODO: venue]. DOI: [TODO]

In an online video-based study (N=35; 12 participants who are blind or visually impaired, BVI), we compared three levels of auditory information communication in a highly automated vehicle: **low** (safety-relevant information only), **medium** (additionally vehicle control and route updates), and **high** (additionally sightseeing and destination information).

### Data (`bvi-auditory-hav.xlsx`)

| Sheet | Content |
|---|---|
| `original` | Raw survey export, one row per participant (N=39; includes the four excluded responses) |
| `demographics` | Demographic columns of the export |
| `cleaned` | Included participants (N=35), wide format; basis for the demographics analysis |
| `Results` | Long format used for the main analysis: 105 rows = 35 participants × 3 conditions |
| `Final` | Post-study questionnaire (one row per participant): reasonability, necessity, intention to use, usefulness ratings, and ratings of the in-vehicle information system |

Key variables in `Results`:

- `UserID` — participant identifier
- `VIP` — visual impairment: `Y` (BVI) / `N` (sighted)
- `ConditionID` / `infoContent` — condition: `1` = `Few` (low), `2` = `Medium`, `3` = `Most` (high information)
- `tlx_mental` — mental subscale of the raw NASA-TLX (1–20)
- `TiA_Trust1/2` — *Trust* subscale of the Trust in Automation questionnaire (Körber, 2018; 1–5), averaged to `overallTiATrust` in the script
- `perceivedSafety1–4` — semantic differentials (−3 to +3; Faas et al., 2020), averaged to `ps_score`
- `SART1–10` — Situation Awareness Rating Technique items (Taylor, 1990); combined to Demand (1–3), Supply (4–7), Understanding (8–10), and SA = U − (D − S)
- `ueqs1–8` — UEQ-S items (1–7; Schrepp et al., 2017); averaged to pragmatic (1–4) and hedonic (5–8) quality
- `informationCommunica` — "All relevant information has been communicated." (1–5)

In `Final`: `reasonable`, `necessary`, `wouldUse` (1–7); usefulness of the information types `startEnd`, `drivingRelated`, `pedestrian`, `route`, `unforeseenEvents`, `poi`, `surroundings` (1–5); rating of the in-vehicle information system `light`, `dashboard`, `sound`, `voice` (1–5).

**Exclusions:** 39 complete responses were collected; four were excluded during data cleaning (one participant failed all three attention checks and reported video-buffering problems). All analyses use N=35.

## Reproducing the analysis

The analyses reported in the paper were run with **R 4.6.0** (packages as of June 2026, ggstatsplot 1.0.0).

```r
install.packages(c("easystats", "ARTool", "dplyr", "tidyr", "ggplot2",
                   "FSA", "rstatix", "readxl", "ggstatsplot", "see", "remotes"))
remotes::install_github("M-Colley/colleyRstats")
```

Then run the script from this directory (it sets its working directory automatically):

```r
source("bvi-auditory-hav.R")
# or from the command line:
# Rscript bvi-auditory-hav.R
```

Statistical pipeline, as described in the paper:

- **Assumption checks** (Shapiro–Wilk on model residuals and by group, Levene) via `colleyRstats::checkAssumptionsForAnova` decide between the parametric and non-parametric path per dependent variable.
- **Non-parametric data** (cognitive load, trust, perceived safety, SART Demand/Understanding, UEQ-S, information presence): Aligned Rank Transform mixed ANOVA (`ARTool`), post-hoc Dunn's tests with Holm correction (`FSA::dunnTest`).
- **Parametric data** (SART SA and Supply): mixed repeated-measures ANOVA (`rstatix::anova_test`).
- **Between-group comparisons** of the post-study questions: Mann–Whitney U tests (via `ggstatsplot`/`statsExpressions`).

The analysis pipeline uses [colleyRstats](https://github.com/M-Colley/colleyRstats) (Colley, 2025; DOI: [10.5281/zenodo.18046754](https://doi.org/10.5281/zenodo.18046754)).
